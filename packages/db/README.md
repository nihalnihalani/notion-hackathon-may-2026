# @forge/db

The Prisma schema and singleton client for Forge's PlanetScale (Postgres) source-of-truth database. Tables: `workspaces`, `users`, `generations`, `generation_steps`, `generated_agents`, `audit_log`, `usage_meter`, `prompt_cache`, `evaluations`, `billing_events`. Exports a single `prisma` client plus typed query helpers.
